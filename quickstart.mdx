## Installation

<CodeGroup>
  ```bash npm
  npm install quokkajs
  ```
  ```bash yarn
  yarn add quokkajs
  ```
</CodeGroup>

quokkajs requires React 18 or higher as a peer dependency.

---

## Step 1: Wrap your app in QuokkaProvider

`QuokkaProvider` sets up the shared in-memory cache and gives hooks access to your global state (for injecting auth tokens, etc.).

```tsx main.tsx
import React from 'react';
import ReactDOM from 'react-dom/client';
import { QuokkaProvider } from 'quokkajs';
import App from './App';

ReactDOM.createRoot(document.getElementById('root')!).render(
  <QuokkaProvider getState={() => {}}>
    <App />
  </QuokkaProvider>
);
```

<Note>
  If you use Zustand or Redux, pass your store's `getState` function — e.g. `useStore.getState` or `store.getState`. This lets you read state (like an auth token) inside `prepareHeaders`. If you don't have a global store, `() => {}` is fine.
</Note>

## Step 2: Define your API

Use `createApi` to describe your endpoints. Each key in `endpoints` becomes a hook.

```ts api/todoApi.ts
import { createApi } from 'quokkajs';

type Todo = {
  id: number;
  title: string;
  isDone: boolean;
};

const todoApi = createApi({
  apiName: 'todoApi',
  baseUrl: 'https://api.example.com',
  tags: ['todos'],
  endpoints: (builder) => ({
    getTodos: builder.query<string, Todo[]>((search) => ({
      url: '/todos',
      params: { search },
    }), { providesTags: ['todos'] }),

    createTodo: builder.mutation<{ title: string }, Todo>((body) => ({
      url: '/todos',
      method: 'POST',
      body,
    }), { invalidatesTags: ['todos'] }),

    deleteTodo: builder.mutation<{ id: number }, void>((args) => ({
      url: `/todos/${args.id}`,
      method: 'DELETE',
    }), { invalidatesTags: ['todos'] }),
  }),
});

export const {
  useGetTodosQuery,
  useCreateTodoMutation,
  useDeleteTodoMutation,
} = todoApi.actions;
```

<Info>
  Hook names are derived automatically from your endpoint keys:
  `getTodos` → `useGetTodosQuery`, `createTodo` → `useCreateTodoMutation`.
</Info>

## Step 3: Use the hooks in your components

```tsx App.tsx
import React from 'react';
import {
  useGetTodosQuery,
  useCreateTodoMutation,
  useDeleteTodoMutation,
} from './api/todoApi';

function TodoList() {
  const [search, setSearch] = React.useState('');
  const { data: todos, loading, error } = useGetTodosQuery(search, {
    fetchOnRender: true,
    fetchOnArgsChange: true,
    debouncedDuration: 300,
  });

  const { trigger: createTodo } = useCreateTodoMutation();
  const { trigger: deleteTodo } = useDeleteTodoMutation();

  if (loading) return <p>Loading...</p>;
  if (error) return <p>Something went wrong.</p>;

  return (
    <div>
      <input
        value={search}
        onChange={(e) => setSearch(e.target.value)}
        placeholder="Search todos..."
      />

      {todos?.map((todo) => (
        <div key={todo.id}>
          <span>{todo.title}</span>
          <button onClick={() => deleteTodo({ id: todo.id })}>Delete</button>
        </div>
      ))}

      <button onClick={() => createTodo({ title: 'New todo' })}>
        Add Todo
      </button>
    </div>
  );
}
```

<Note>
  When `createTodo` or `deleteTodo` succeeds, `useGetTodosQuery` **automatically re-fetches** because their `invalidatesTags` overlaps the query's `providesTags`. No manual refetch needed.
</Note>
